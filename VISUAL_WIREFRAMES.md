## ANIMATION TIMING REFERENCE

### Standard Durations
- Micro-interactions: 200ms
- Card transitions: 300ms
- Section reveals: 600-800ms
- Page transitions: 400ms
- Counter animations: 2000ms

### Standard Easings
- UI interactions: `cubic-bezier(0.4, 0, 0.2, 1)` (ease-out)
- Reveals: `cubic-bezier(0.16, 1, 0.3, 1)` (expo-out)
- Bounce: `cubic-bezier(0.34, 1.56, 0.64, 1)` (spring)
- Smooth: `cubic-bezier(0.65, 0, 0.35, 1)` (ease-in-out)

### Stagger Delays
- Text lines: 100ms
- Cards: 150ms
- Table rows: 80ms
- List items: 50ms
